## 1. 项目概览

| 字段 | 值 |
|------|------|
| **项目名称** | 幸运转盘（我替你选择 / my-lucky-wheel） |
| **项目类型** | 纯前端静态 Web 应用 |
| **技术栈** | React 19 + TypeScript 5 + Vite 6 + Tailwind CSS v4 + shadcn/ui（Radix UI） |
| **图标库** | Lucide React |
| **部署方式** | Docker（多阶段构建：Node 构建 → nginx 托管静态文件） |
| **语言** | 中文（zh-CN） |
| **设计方案** | [`docs/重构设计方案.md`](重构设计方案.md) |

### 功能简介

一个幸运转盘随机选择工具，用户可以：

- 自定义转盘项目（最多 16 个）
- 旋转转盘获取随机结果
- 设置每日使用次数限制（1~100 次）
- 查看历史选择记录（最多保存 100 条）
- 鼠标悬停转盘扇形可显示完整项目名称（Tooltip）

## 3. 关键文件说明

### `src/main.tsx` — 入口文件
- 使用 `createRoot` 挂载 React 应用到 `#root`
- 引入全局样式 `index.css`

### `src/App.tsx` — 根组件
- 管理全局状态：`items`（项目列表）、`history`（历史记录）、弹窗开关状态
- 组合所有子组件：Header + Main(Wheel + ItemList) + Modals + Footer
- 弹窗：ConfigModal（配置）、HistoryModal（历史）、ResultModal（结果）
- 使用 `useWheelSpin` hook 驱动转盘旋转

### `src/components/Wheel.tsx` — 转盘组件
- 使用 Canvas 2D API 绘制转盘
- 高 DPI 屏幕适配（devicePixelRatio）
- 16 种黑白灰 GitHub 风格配色
- 鼠标悬停 Tooltip（根据角度计算对应扇形索引）
- 指针使用 CSS border 三角形实现

### `src/hooks/useWheelSpin.ts` — 旋转动画 Hook
- 使用 `requestAnimationFrame` 实现平滑动画
- `easeOutCubic` 缓动函数
- 至少旋转 5 圈，3 秒动画时长
- 自动清理 animationFrame（`cancelAnimationFrame`）

### `src/hooks/useUsageLimit.ts` — 使用限制 Hook
- 跨天自动重置使用次数
- 旧版 localStorage 数据兼容迁移（`limitEnabled` → `wheelLimitSettings`）

### `src/index.css` — 全局样式
- Tailwind CSS v4（使用 `@theme` 定义自定义颜色变量）
- GitHub 黑白风格主题色
- `body` 使用 flex 布局保证 footer 底部对齐

## 4. 数据存储（localStorage）

| Key | 类型 | 说明 |
|-----|------|------|
| `wheelItems` | `string[]` | 转盘项目列表 |
| `wheelHistory` | `HistoryItem[]` | 历史记录（含 id/name/date/time） |
| `wheelLimitSettings` | `LimitSettings` | 使用限制设置（enabled/dailyLimit/usedCount/lastDate） |

兼容旧版 key：`limitEnabled`、`dailyLimit`、`usedCount`、`lastDate`（首次加载自动迁移）

## 5. 设计风格

- **GitHub 黑白风格**：深色 header（`#24292f`）、浅灰背景（`#f6f8fa`）、白色卡片
- **字体**：系统字体栈（-apple-system / BlinkMacSystemFont / Segoe UI / Noto Sans SC）
- **转盘配色**：16 种黑白灰色调，白色分隔线
- **组件**：shadcn/ui（基于 Radix UI，黑白主题定制）
- **图标**：Lucide React（替代内联 SVG）

---

## 6. 开发约定

- 使用 **函数组件 + Hooks**，不使用 class 组件
- 使用 **TypeScript** 严格模式
- 使用 **Tailwind CSS v4** 原子化样式，不写自定义 CSS 类（index.css 除外）
- shadcn/ui 组件直接放在 `src/components/ui/` 下，不使用 CLI 安装
- 中文 UI 文案，代码注释使用中文
- `@/` 别名指向 `src/` 目录
- 无路由、无状态管理库
