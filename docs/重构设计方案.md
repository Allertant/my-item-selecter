## 3. 组件架构

```
App
├── Header                        # 顶部导航栏
├── main
│   ├── Wheel                     # Canvas 转盘（含内联指针 + Tooltip）
│   └── ItemList                  # 项目列表 + 添加/删除/配置/历史入口
├── ConfigModal                   # 配置弹窗（Dialog + Switch + Input）
├── HistoryModal                  # 历史记录弹窗（Dialog）
├── ResultModal                   # 结果弹窗（Dialog）
└── Footer                        # 页脚
```

### 自定义 Hooks

| Hook | 说明 |
|------|------|
| `useLocalStorage<T>` | 泛型 localStorage 读写，自动 JSON 序列化 |
| `useWheelSpin` | 转盘旋转动画（requestAnimationFrame + easeOutCubic） |
| `useUsageLimit` | 使用限制（跨天重置 + 旧版数据迁移） |

---

## 4. 数据存储

保持 localStorage，新增旧版数据兼容迁移：

| Key | 类型 | 说明 |
|-----|------|------|
| `wheelItems` | `string[]` | 项目列表 |
| `wheelHistory` | `HistoryItem[]` | 历史记录（含 id） |
| `wheelLimitSettings` | `LimitSettings` | 限制设置（统一存储） |

旧版 key（`limitEnabled`/`dailyLimit`/`usedCount`/`lastDate`）首次加载自动迁移至 `wheelLimitSettings`。
